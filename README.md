# 🌱 Missão Agro Sustentável 2026

Projeto educativo desenvolvido para o **Concurso Agrinho 2026**, com o objetivo de demonstrar como a tecnologia pode fortalecer a conexão entre o campo e a cidade, promovendo sustentabilidade, inovação e consciência ambiental.

---

## 📌 Objetivo do Projeto

A plataforma busca conscientizar estudantes e a comunidade sobre:

- 🌾 A importância da agricultura sustentável  
- 🏙️ A relação entre campo e cidade  
- 💧 O uso consciente da água  
- 🤖 O impacto da tecnologia no campo  
- ♻️ Práticas sustentáveis no dia a dia  

---

## 🚀 Funcionalidades

### 🎮 Mini Game Educativo
Simula o ciclo de produção agrícola:
- Plantio 🌱
- Irrigação 💧
- Colheita 🌾
- Venda para a cidade 🏙️

---

### 🧠 Quiz Interativo
Quiz com 10 perguntas sobre:
- Agricultura
- Sustentabilidade
- Tecnologia no campo
- Recursos naturais

---

### 💧 Calculadora de Economia de Água
Permite calcular o consumo de água com base em:
- Área plantada
- Tipo de irrigação

---

### 🔄 Simulador Campo x Cidade
Mostra a interdependência entre:
- Produção rural
- Serviços urbanos

---

### 📊 Gráficos Interativos
Utilizando Chart.js:
- Produção sustentável
- Economia de água

---

### ♿ Acessibilidade
O projeto inclui recursos de acessibilidade:
- Alto contraste
- Leitura de tela (voz)
- Modo sem imagens
- Controle de animações
- Integração com VLibras

---
